1Request parameter SyncLockRecordAction

attribute type illustrate
startNum int Read the starting index of the record (note: the index of the latest record is 0)
readCnt int Read quantity, usually filled in 10
logVersion int Door lock record version
1: represents the first generation door lock operation record
2: represents the second generation door lock operation record
Note: According to the menuFeature in the door lock DNA information, the third bit is 1, indicating that the door lock only supports the second generation operation record. Otherwise, it indicates that the door lock only supports the first generation operation record
